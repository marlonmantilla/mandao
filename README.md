# Mandao

Mandao gem allows developers to access Click2Mail REST services in a simple way.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mandao'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mandao
